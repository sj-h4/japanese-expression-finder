## テキストの事前処理

`nkf` がインストールされていれば

```=bash
bash convert.sh example.txt
```

で `example.txt` を UTF-8 に変換したファイルが `output/` 以下に置かれます（`output/` フォルダがない場合は作成してください）。

## 方針

- 一行ずつ形態素解析して、「には」か「は」や「も」の前後数語以内に同じ語が使われているかを調べる

### 問題点

- 「やるにはやる」という文は形態素解析の結果「やる」「に」「はやる」となるため、抽出されない
- 「仕事は仕事だ」のような表現も引っかかる
